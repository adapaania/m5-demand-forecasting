# M5 Demand Forecasting

End-to-end demand forecasting project using the M5 Forecasting dataset.

## Objectives
- Understand demand patterns: trends, seasonality, anomalies
- Compare classical, additive, and deep learning models
- Analyze trade-offs between interpretability and accuracy

## Dataset
- M5 Forecasting (Walmart sales)

## Models (planned)
- ARIMA / SARIMA
- Prophet
- LSTM / Deep Learning
